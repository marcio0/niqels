{% trans "Movimentações "%}

{% trans "Criando uma movimentação" %}

{% blocktrans %} Para criar uma movimentação, use o formulário à esquerda. Preencha a categoria e valor, e caso ache necessário, a descrição e data (por padrão é o dia de hoje).
Ao criar uma movimentação, ela será adicionada à listagem ao lado, e seu balanço será atualizado com os novos dados. {% endblocktrans %}

{% trans "Categorias de movimentações" %}

{% blocktrans %} As categorias servem para definir o tipo de cada movimentação. Elas ajudam a mostrar para onde seu dinheiro está indo, e são vistas com clareza nos relatórios.
{% endblocktrans %}

{% blocktrans %} Para buscar uma categoria, basta digitar no campo ao topo da listagem, e apenas as categorias relevantes serão mostradas. {% endblocktrans %}

{% trans "Criando uma movimentação parcelada" %}

{% blocktrans %} Uma movimentação parcelada se divide igualmente entre o número de meses especificado. Por exemplo, ao parcelar em 5 vezes uma compra no valor de R$ 550,00, serão criadas 5 movimentações de R$ 110,00, sendo a primeira no dia especificado, e as outras nos meses seguintes, sempre no mesmo dia do mês.
Para criar uma movimentação parcelada, marque a caixa de seleção "Esta é uma compra parcelada". Campos adicionais irão surgir para especificar quantidade e valor das parcelas. {% endblocktrans %}

{% blocktrans %} Quando uma movimentação é uma parcela, é exibido na listagem de movimentações o número desta parcela: {% endblocktrans %}

{% trans "Editando uma movimentação" %}

{% blocktrans %} Para editar uma movimentação, basta clicar nela na listagem. Uma janela se abrirá com o formulário de edição. Preencha com os novos valores e clique em “Salvar”. {% endblocktrans %}

{% trans "Removendo uma movimentação" %}

{% blocktrans %} Para remover uma movimentação, pare o cursor em cima dela. Um ícone de lixeira aparecerá ao lado do valor. Clique nela para remover. {% endblocktrans %}

{% blocktrans %} Também é possível remover a movimentação através da janela de edição. {% endblocktrans %}

{% endblock %}
